## Installation

Firstly navigate to the project directory using your terminal. 

*For example :*
`cd Desktop/flappy_birds`

Then based upon your operating system follow the steps below

### Windows
`build.bat`

### Linux
`chmod +x build.sh`

`./build.sh`
